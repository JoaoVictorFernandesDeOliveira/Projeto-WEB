API do Banco Central para criar um gráfico que mostra a projeção da inflação para os próximos meses. A ideia é fazer com que o usuário consiga ver de um jeito simples se os preços das coisas tendem a subir ou a cair, ajudando ele a se planejar melhor e a cuidar do seu dinheiro.
